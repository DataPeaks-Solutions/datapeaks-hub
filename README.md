## Access model

**Level 1 of every course is fully free** — all concepts, all tools, no paywall. Levels 2 & 3 go deeper and are part of paid DataPeaks programs.

Project code in this hub uses only open-source / publicly available datasets. Proprietary student datasets and paid-course materials are not published here.

## How this hub works

Each course has 3 levels — Beginner, Intermediate, Advanced. One project per week, building toward placement-level mastery of every tool in that stack. Start at Level 1 of any track, or follow the weekly drop in order.
